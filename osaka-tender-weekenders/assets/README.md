# assets — 画像の配置先

`index.html` は下記のファイル名で画像を参照します。
ここに同名で画像を置けば、自動的にLP上へ表示されます（未配置時はラベル付きプレースホルダ表示）。

| ファイル名 | 用途 | 表示サイズ（目安） |
|---|---|---|
| `otw-watanai.jpg`   | 綿内克幸 アーティスト写真 | 横100% × 高さ300px |
| `guest-nishimura.jpg` | 西村哲也（ゲスト）正方形 | 150 × 150px |
| `guest-bookends.jpg` | ブックエンズ ライブ写真 | 横100% × 高さ210px |
| `guest-troy.jpg`    | トロイと僕らは ライブ写真 | 横100% × 高さ210px |
| `venue-1.jpg`       | 雲州堂 店内（大） | 横100% × 高さ240px |
| `venue-2.jpg`       | 店内 / 外観 | 約半幅 × 高さ130px |
| `venue-3.jpg`       | ステージ | 約半幅 × 高さ130px |

- 表示は `object-fit: cover`。多少アスペクト比が違ってもトリミングされて収まります。
- 推奨: 表示サイズの2倍（Retina対応）程度の解像度。容量は各 200〜400KB を目安に最適化。
- 形式は jpg を想定していますが、png/webp を使う場合は `index.html` の `src` を合わせて変更してください。

参考ポスター（一次情報）は handoff バンドル内 `assets/poster_reference.jpg` を参照。
